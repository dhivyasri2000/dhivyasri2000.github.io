Projects

A real-time System for Interpretation of Indian Sign Language to aid physically challenged people

A Machine Learning based system that translates Indian Sign Language (ISL) hand gestures to text and vice versa. This system enables the people with hearing and speech impairments to communicate effectively with the general public without the need for human sign language translators. The proposed automated sign language translation was achieved using image processing techniques and machine learning algorithms. Hand gesture data was captured using a camera, pre-processed, and fed into a trained Neural Network Classifier that identifies the meaning of the gestures and displays the translated text. This is a real-time system that continuously takes-in hand gestures as video input, and displays the corresponding translated words on the same video screen.

Whisper Detection

This project based on detection of whispered speech was done as a part of Samsung PRISM Internship with Samsung’s Voice Intelligence R&D Team. This whisper detection system was aimed at building specialized ASR systems for understanding whispers with better accuracy. A dataset consisting of both normal and whispered speech was collected. Feature extraction was done using MFCC (Mel Frequency Cepstral Coefficients). The CRNN (Convolutional Recurrent Neural Network) based classifier model was trained using the pre-processed dataset. The developed model was able to detect whispers and convert them to text.

Automated Toll Plaza

Automated Toll Fee Collection system for collecting toll charges using RFID Technology to avoid manual money transactions which in turn reduces traffic jams at the toll gates, and saves time. This project aided in having practical experience on designing digital circuits using micro-controllers to solve real-world problems. This system is based on RFID Technology. Every vehicle has a tag with a unique ID. An RFID reader is placed at the toll plaza. When the vehicle passes through the plaza, the tax amount is deducted from the owner’s prepaid balance. The system also contains provisions for identifying stolen vehicles using the tag ID
